## Core Features:

- Case Entry: Manual input form for new cases of corruption with fields for the accused, accusations, sentences, and political allegiance categories.
- Data Import Automation Tool: An AI tool to automatically extract and categorize case data from mainstream media and official legal sources. This includes identifying key figures, accusations, sentences, and political allegiance, ensuring timely leaderboard updates.
- Scheduled Updates: Implementation of a scheduled task to refresh and update the leaderboard data, ensuring new information is available on the website according to schedule (e.g., every 8 hours).
- Leaderboard Display: Interactive table to display cases, sortable by accusations, sentences, or category. Rows include links to sources, when available.
- Political Allegiance Categorization: Assign each case to a political allegiance category: member of government, family member, public servant, or institution. Enables users to filter cases by political affiliation.

## Style Guidelines:

- Primary color: Dark crimson (#8B0000). The primary color represents corruption through its similarity to blood, is vibrant but professional, and still has enough brightness to contrast well with a dark background.
- Background color: Very dark gray (#222222). This color establishes the dark scheme and a serious tone appropriate for this content.
- Accent color: Dark orange (#D4A373). This color provides contrast and highlights points of user interaction. It's brightness and saturation is clearly distinct from the primary and background colors, while remaining within the chosen analogous hue range.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look. 
- The leaderboard should be structured as a single-page application with the table taking center stage, optimized for both desktop and mobile viewing. Sticky headers to persist sortability.
- Simple, universally-understood icons to represent the different political allegiance categories and types of evidence or sources.
- Subtle transition animations for sorting and filtering updates to enhance the user experience without distracting from the data.
